wget 下载
=========
>请提前安装wget
>使用以下命令下载脚本
```shell
wget https://raw.githubusercontent.com/coding1618/Google-images-shell/master/Google_images.sh
```

给你脚本添加执行权限
==================
```shell
  chmod +x ./Google_images.sh
```

好了执行完脚本按照提示操作~Good luck!
========================

>安装成功之后记得开启防火墙端口哦~
>或者关闭防火墙，下面是centos7关闭防火墙的命令
```shell
systemctl stop firewalld
```
